Fiche JDR

Ceci est une application destinée à gérer simplement une fiche personnage de JDR depuis un mobile avec une base de stats notées en pourcentages.

Utilisez le botton suppérieur gauche pour basuler entre les modes "Edition", "Jouer" et "Suppression".

En mode Edition, cliquez sur "+" pour ajouter une stat ou sur une stat pour la modifier.

En mode "Jouer", cliquez sur une stat ou sur un dé en bas de page pour effectuer un jet.

En mode "Suppression, cliquez sur une stat pour la retirer. Attention, cette opération est irrévesible.
